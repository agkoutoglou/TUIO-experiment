TUIO Example
============

In the process to get familiar with TUIO protocol, this is a project in Processing language, to experiment with image processing. Below follows the list of the fiducials used and their functions:

Fiducial 1 - Adjusts length of the image by rotation
Fiducial 2 - Adjusts height of the image by rotation
Fiducial 3 - Adjusts size of the image by rotation
Fiducial 5 - Inserts an image in the x-y coordinates.
Fiducial 5 - Inserts another one image in the x-y coordinates.
Fiducial 10 - Adjusts length of the image by rotation
